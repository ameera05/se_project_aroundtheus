#Sprint 3 Project - Around The U.S.

##Overview:
* Responsive Web Design for Mobile, Desktop & Tablet
* Understanding Figma
* Connecting Fonts
* Implementing Grid Layout

##Intro
The purpose of this project is to understand responsive web design for popular screen sizes. We also are introduced to Figma and how to gather essential information there and implement on the front-end.
